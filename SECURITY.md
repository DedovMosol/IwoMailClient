# Security Policy

## Supported versions

Only the latest released version receives security fixes. Please update before
reporting an issue.

| Version | Supported |
|---------|-----------|
| 1.6.x   | ✅        |
| < 1.6   | ❌        |

## Reporting a vulnerability

**Please do not open a public GitHub issue for security problems.**

Since this app handles mail account credentials, EAS/EWS traffic and local
message storage, responsible disclosure is appreciated. Report privately:

- **Email:** andreyid@outlook.com
- **Telegram:** [@i_wantout](https://t.me/i_wantout)

Please include:

- affected version and Android version,
- steps to reproduce,
- impact assessment (what data / account is exposed).

You can expect an initial response within a few days. Fixes for confirmed
issues are shipped in the next patch release, with credit if you wish.
